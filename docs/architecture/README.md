## 核心组件

| 组件 | 目录 | 语言 | 职责 |
|------|------|------|------|
| **App** | `app/` | TypeScript/React Native | 用户界面 |
| **App-Bridge** | `app-bridge/` | TypeScript | 客户端通信库 |
| **Daemon** | `daemon/` | Go | 核心服务 |
| **Relay** | `relay-go/` | Go | 连接中继 |
| **CLI** | `cli/` | Go | 命令行工具 |
| **Protocol** | `protocol/` | Go | 协议定义 |
